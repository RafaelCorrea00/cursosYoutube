# Projeto 1 - Segurança Virtual

- Você precisa pegar o nome da pessoa que está tentando entrar naquele lugar

- Você precisa perguntar a ele(a) quem a convidou para essa festa

- Você precisa de um botão na tela que quando você clicar irá acessar o seu código javascript para verificar se a pessoa que está tentando entrar foi de fato convidado por quem ela diz que convidou.

- Ex.: Nome da pessoa: 'Jhonatan'.
- Quem o convidou para festa? 'Cristian'.
- Clico no botão, o javascript pega as informações que eu preenchi na tela e verifica no meu código javascript se o nome da pessoa está na lista de convidados.

- Caso o nome esteja na lista, alterar a descrição de 'Pesquisando' para 'Você pode entrar'
- Caso o nome não esteja na lista, alterar a descrição de 'Pesquisando' para 'Você não pode entrar'
